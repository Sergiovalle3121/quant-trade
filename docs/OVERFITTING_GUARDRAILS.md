# Overfitting Guardrails

Campaign ranking is deliberately conservative. It requires out-of-sample metrics, benchmark comparison, cost sensitivity, enough trades, complete artifacts, and multi-component scoring.

Penalties are applied for high train/test performance gaps, excessive turnover, and high drawdown. Single-metric ranking is prohibited because it encourages false precision and fake-profit optimization.

These checks are research controls only and do not approve real-money trading.
