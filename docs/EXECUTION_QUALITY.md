# Execution Quality Lab

The execution quality lab simulates research-only execution models: `next_open`, `next_close`, `vwap_proxy`, `spread_adjusted_open`, `volume_participation_limited`, `partial_fill_model`, and `adverse_gap_model`.

Conservative assumptions are defaults. Partial fills are limited by historical bar volume participation. No live endpoints, secrets, broker adapters, or real order routing are used.
