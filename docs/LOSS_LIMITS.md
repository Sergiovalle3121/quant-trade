# LOSS LIMITS

This Phase 19 artifact is paper-only governance analysis. It does not approve real-money trading, does not enable live broker endpoints, and does not place real orders.

Paper performance does not guarantee real results. Any real-money decision would require a separate explicit human, legal, compliance, and risk process outside this codebase.

The codebase still blocks live endpoints and real-money approval. Outputs must keep real_money_ready=false and real_money_approved=false.
